# Cloud DevOps AI Security Platform

DevOps 기반 클라우드 보안 실습 및 AI 기반 공격 분석 플랫폼

## 주요 기능

- Lab 환경 자동 생성
- 공격 실습 환경 제공
- 로그 수집 및 분석
- AI 기반 공격 탐지
- AI 기반 공격 설명
- 실시간 보안 알림
- 자동 대응 기능

## 기술 스택

Frontend: React  
Backend: FastAPI  
Infrastructure: Terraform  
Container: Docker  
Logging: ELK Stack  
AI: Python / Scikit-learn / LLM
